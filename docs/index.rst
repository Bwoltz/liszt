Welcome to Quick Note's documentation!
==============================================

**Quick Note** is a command-line tool designed to
make taking small notes from the command line as
easy as possible.

Installation
------------

Install **Quick Note** by first running ``brew tap scamacho23/quicknote``.
Next, run ``brew install qnote``. Start using ``qnote`` by running ``qnote -h``
or ``qnote -help`` to view the available commands.

Contribute
----------

- Make issues at https://github.com/scamacho23/homebrew-quicknote/issues
- Source code held at https://github.com/scamacho23/homebrew-quicknote

Support
-------

If you are having issues with your version of **Quick Note**,
contact us at quicknote.v1@gmail.com. Make sure to update
**Quick Note** relatively frequently by running ``brew update``
and then ``brew reinstall qnote``.
*To view your version of Quick Note, run qnote -version or qnote -v*
